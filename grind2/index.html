Гайд на гринд v2

Чистая версия без старого кэша. Не финансовая рекомендация.

Нет моей точки = нет сделки.

Стратегия = шаблон. Сделка = конкретный вход.

Как работать

  1. Открой «Стратегия».
  2. Выбери тип и нажми «Заполнить пример».
  3. Нажми «Сохранить стратегию».
  4. В сохранённых нажми «Повторить в сделке».
  5. В сделке допиши тикер, уровень, стоп и потенциал.

На главный экран

Chrome → ⋮ → Добавить на главный экран / Установить приложение.
